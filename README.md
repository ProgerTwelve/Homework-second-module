# Цель проекта
Данное приложение создано для обработки банковских операций.
Приложение позволяет фильтровать операции по статусу и сортировать их по дате

## Описание функций
1. Модуль masks.py

* Функция get_mask_card_number - Маскирует номер банковской карты в формате 0000 00** **** 0000
Принимает на вход строковые и числовые значения, возвращает строковые значения
* Функция get_mask_account - Маскирует номер банковского счета в формате **0000
Принимает на вход строковые и числовые значения, возвращает строковые значения

2. Модуль processing.py
* Функция filter_by_state - Функция возвращает новый список словарей, содержащий только те словари, у которых ключ
state соответствует указанному значению
Принимает на вход список словарей
Примеры данных: [{'id': 41428829, 'state': 'EXECUTED', 'date': '2019-07-03T18:35:29.512364'}, {'id': 939719570, 'state': 'EXECUTED', 'date': '2018-06-30T02:08:58.425572'}, {'id': 594226727, 'state': 'CANCELED', 'date': '2018-09-12T21:27:25.241689'}, {'id': 615064591, 'state': 'CANCELED', 'date': '2018-10-14T08:21:33.419441'}]
* Функция sort_by_date - Возвращает новый список, отсортированный по дате
Примеры данных: [{'id': 41428829, 'state': 'EXECUTED', 'date': '2019-07-03T18:35:29.512364'}, {'id': 939719570, 'state': 'EXECUTED', 'date': '2018-06-30T02:08:58.425572'}, {'id': 594226727, 'state': 'CANCELED', 'date': '2018-09-12T21:27:25.241689'}, {'id': 615064591, 'state': 'CANCELED', 'date': '2018-10-14T08:21:33.419441'}]

3. Модуль widget.py
* Функция mask_account_card
Принимает строку с информацией о типе и номере карты или счета
    и возвращает замаскированный номер карты
Аргументом может быть строка типа 
Visa Platinum 7000792289606361
, или 
Maestro 7000792289606361
, или 
Счет 73654108430135874305
* Функция get_date - Возвращает строку с датой в формате 'ДД.ММ.ГГГГ
Принимает на вход строку с датой в формате 
"2024-03-11T02:26:18.671407"
 и возвращает строку с датой в формате 
"ДД.ММ.ГГГГ"
 (
"11.03.2024"
).
